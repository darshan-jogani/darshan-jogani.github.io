# Darshan Jogani — Academic Portfolio

Welcome to the repository for my personal academic and professional portfolio. 

**Live Website:** [https://darshan-jogani.github.io](https://darshan-jogani.github.io)

## Repository Structure

To keep deployments clean and the source code organized, this repository utilizes specific branches for different purposes:

- **`master`**: You are here! This branch only serves as the repository's entry point and contains this README.
- **`react-source`**: **The active source code.** Built with React, Vite, Three.js, and GSAP. Switch to this branch to view or edit the code.
- **`gh-pages`**: The compiled, production-ready build that is automatically served by GitHub Pages.
- **`legacy-jekyll`**: An archive branch containing the previous, Jekyll-based version of this portfolio.

## Exploring the Code

To view the code or run the project locally, switch to the `react-source` branch:

```bash
git checkout react-source
npm install
npm run dev
```
